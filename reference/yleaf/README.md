# Yleaf Reference Code

This directory contains reference code from Yleaf v3.2.1 for understanding
the existing algorithm and data structures.

See the main project files at: https://github.com/genid/Yleaf

Key files to reference:
- predict_haplogroup.py - Haplogroup prediction algorithm
- Yleaf.py - Main pipeline
- tree.py - Tree data structure

The source is provided in the RESEARCH.md document as context.
